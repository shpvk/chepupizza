ChepuPizza is a pizza ordering web application with ready-made pizzas, a custom pizza constructor, cart functionality, user authentication, and order creation.

## Live Demo

Frontend: https://chepupizzeria.vercel.app/
API: https://chepupizzeria.onrender.com
Swagger: https://chepupizzeria.onrender.com/swagger

## Tech Stack

**Frontend**

* React
* Vite
* JavaScript
* CSS
* Vercel

**Backend**

* ASP.NET Core Web API
* C#
* Entity Framework Core
* PostgreSQL
* JWT Authentication
* Docker
* Render

**Database & Storage**

* Supabase PostgreSQL
* Supabase Storage for pizza and ingredient images

## Features

* View ready-made pizzas
* Build a custom pizza with ingredients
* Add pizzas to cart
* Register and log in
* Create orders
* Store images in Supabase Storage
* REST API with layered backend architecture

This project was created as a full-stack pet project to practice backend development with ASP.NET Core, Entity Framework Core, PostgreSQL, authentication, deployment, and frontend integration with React.
